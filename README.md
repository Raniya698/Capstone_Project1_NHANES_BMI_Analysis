# NHANES BMI Analysis

## About the Project

This project was completed as part of my capstone work to understand and analyze the NHANES 2020 Body Measurement dataset. The main objective was to explore BMI-related information, clean the dataset, perform data analysis, and visualize the results using Python.

Working on this project helped me improve my understanding of data preprocessing, exploratory data analysis, and data visualization techniques.

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Dataset

The project uses the NHANES 2020 Body Measurement dataset, which contains body measurements collected for health and nutrition research.

## Project Workflow

- Imported and explored the dataset
- Cleaned missing and unnecessary data
- Performed exploratory data analysis
- Visualized BMI distribution and other observations
- Summarized the findings

## Results

The analysis provided useful insights into BMI distribution within the dataset and demonstrated how data preprocessing and visualization can help understand real-world health data.
